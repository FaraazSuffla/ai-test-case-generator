# Example Outputs

This directory contains example test files generated by the AI Test Case Generator.

## Files

| File | Format | Description |
|------|--------|-------------|
| `test_login_playwright.py` | Playwright (Python) | 14 tests covering login functionality across all 4 categories |
| `login.feature` | Gherkin | 12 scenarios with Scenario Outlines for the same login page |

## How These Were Generated

```bash
# Playwright tests
python generate_tests.py --url https://example.com/login --format playwright

# Gherkin feature file
python generate_tests.py --url https://example.com/login --format gherkin
```

## Key Observations

The AI-generated tests include coverage areas that manual test writing often misses:

- **Security tests**: SQL injection, XSS payloads, brute force protection
- **Boundary values**: Max-length emails (254 chars), min/max password lengths
- **Unicode handling**: International characters in input fields
- **Rate limiting**: Multiple rapid failed attempts
- **Session management**: Remember me, redirect after login

All tests use modern Playwright conventions (role-based selectors, `expect` assertions)
and follow pytest best practices (class grouping, descriptive docstrings).
